# ✨ TODO App

モダンなデザインの高機能 TODO 管理アプリケーション

React + TypeScript + Vite + Tailwind CSS で構築された、グラスモーフィズムデザインが特徴の TODO アプリです。

## 🎨 特徴

- **モダンなデザイン**: グラスモーフィズム効果とグラデーションを使用した美しいUI
- **完全な機能**: 追加、編集、削除、完了切り替え
- **高度な分類**: カテゴリとタグで TODO を整理
- **優先度管理**: 3段階の優先度設定（低・中・高）
- **期限管理**: 期限設定と期限切れアラート
- **フィルタリング**: ステータス、カテゴリ、タグ、優先度で絞り込み
- **ソート機能**: 作成日、優先度、期限でソート
- **データ永続化**: ブラウザの localStorage に自動保存
- **レスポンシブ**: デスクトップ・タブレット・モバイル対応

## 🚀 技術スタック

- **React 18** - UI ライブラリ
- **TypeScript** - 型安全な開発
- **Vite** - 高速ビルドツール
- **Tailwind CSS** - ユーティリティファーストCSS
- **React Context API** - 状態管理

## 📦 セットアップ

### 必要な環境

- Node.js 18.x 以上
- npm 9.x 以上

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/sakairi-honest/todo-app.git
cd todo-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

開発サーバーが起動したら、ブラウザで http://localhost:5173/ にアクセスしてください。

## 🛠️ 利用可能なコマンド

```bash
# ビルドのプレビュー

```

## 📖 使い方

### TODO の追加

1. 「新しいTODOを追加」フォームにタイトルを入力
2. 必要に応じて説明、優先度、期限を設定
3. カテゴリやタグを選択（新規作成も可能）
4. 「TODOを追加」ボタンをクリック

### TODO の管理

- **完了/未完了の切り替え**: チェックボックスをクリック
- **編集**: 「編集」ボタンをクリックして内容を変更
- **削除**: 「削除」ボタンをクリック

### フィルタリングとソート

- 右側のフィルターパネルでステータス、カテゴリ、タグ、優先度を絞り込み
- TODOリストの「並び順」ドロップダウンでソート順を変更

### カテゴリ・タグの管理

- 「カテゴリ・タグ管理」パネルを開く
- 新しいカテゴリやタグを追加
- 不要なものは削除可能

## 💾 データの保存

すべての TODO データは自動的にブラウザの localStorage に保存されます。
ページをリロードしても、ブラウザを閉じても、データは保持されます。
